# Multiple Choice API
<img alt="release" src="https://img.shields.io/badge/release-alpha-red"><br>
A module that is meant to be used with the installing/modifying/configuring a variety of programs in an intuitive way.

## Features
- Allows vim-keys for keyboard interaction
- Highlighting using the `rich` python module
